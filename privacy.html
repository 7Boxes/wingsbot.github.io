Privacy Policy

Effective Date: July 13, 2026

1. Data We Collect

Operating this application necessitates processing explicit data parameters provided via the Discord API. This data includes, but is not restricted to:

  • Discord User Identifiers (User IDs) and associated Usernames.
  • Guild/Server Identifiers (Guild IDs) and channel mapping objects.
  • Role Identifiers for tracking automated role synchronization and assignment.
  • Roblox Username for verifying your player status.

2. How We Use Data

Collected data structures are strictly leveraged to maintain functional operations within the application environment, specifically:

  • Processing verification protocols linking external accounts to internal database tables.
  • Managing dynamic, automated chat logging and moderation parameters.
  • Displaying interface statistics within dedicated monitoring control panels.

3. Data Storage and Security

All structured records are maintained inside a localized relational database architecture (such as SQLite) hosted on a secure infrastructure. We enforce access control lists to prevent unauthorized administrative operations or external network exploitation.

4. Data Retention and Erasure

Data objects persist for the duration that the application remains joined to your respective guild or as long as your accounts are flagged as active within our verification engine. Users retain an absolute right to execute erasure protocols. Data deletion can be initiated by removing the application from the server or submitting a formal deletion request via our internal administrative channels.

5. Third-Party Disclosures

We do not broker, sell, transfer, or disclose active user records to external marketing networks or third-party corporate entities. Data configurations are strictly used for native operations within the verified Discord application loop.

This application is an independent software deployment and is not officially endorsed by, affiliated with, or maintained by Discord Inc.
